# 🎬 IMDB Movie Review Sentiment Analysis using SimpleRNN

This project is a **Deep Learning based Sentiment Analysis Web App** built using **SimpleRNN and Streamlit**.
It predicts whether a movie review is **Positive or Negative** and also shows the **Prediction Score**.

---

# 🚀 Live Demo

https://moviereviewsimplernn.streamlit.app/

User enters a movie review → Model predicts:

* Sentiment: Positive / Negative
* Prediction Score (Confidence)

---

# 🧠 Model Details

* Dataset: IMDB Movie Reviews Dataset
* Model: SimpleRNN
* Framework: TensorFlow / Keras
* Activation Function: ReLU / Sigmoid
* Padding Length: 500
* Output: Binary Classification
